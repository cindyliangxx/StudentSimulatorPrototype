# 主游戏 UI 素材与文字规范

## 基准画布

- 标准分辨率：1920 x 1080，横屏 16:9。
- Unity Canvas：Screen Space Overlay，CanvasScaler Reference Resolution = 1920 x 1080。
- 整体风格：浅米白背景、深灰文字、浅暖灰描边、低饱和灰绿强调。

## 背景素材

- 当前文件：`Assets/ResourceLibrary/MainGameUI/Backgrounds/spr_ui_game_bg_warm_main.png`
- 图片规格：1920 x 1080 px。
- 格式：PNG，Sprite (2D and UI)。
- 视觉要求：浅米白主色，允许极轻微暖灰纸感或渐变，不使用高饱和色块。
- 主色参考：`#F4F0E8`，底部暖灰过渡参考 `#F0EAE0`。

## 卡片图片素材

- 图片文件必须是正方形 PNG。
- 建议规格：1024 x 1024 px。
- 实际可见内容建议放在中央 1024 x 768 px 或更小区域，其余区域使用透明图层补足。
- Unity 显示区域：约 560 x 420 px，Preserve Aspect 开启。
- 当前无正式素材时使用程序化圆角占位：填充 `#E8E1D6`，描边 `#C9C0B2`，圆角 20 px，描边 2 px。
- 命名示例：`spr_card_daily_001_morning_lecture_main.png`。

## UI 图标素材

- 顶部状态图标使用 512 x 512 px 正方形 PNG，透明背景，存放在 `Assets/ResourceLibrary/MainGameUI/Resources/UI/Stats`。
- 顶部状态填充使用同目录下的 `_fill_mask_512.png` 自动生成 mask 图，保持 512 x 512 px 正方形 PNG。
- 后续替换正式状态图标时，图片文件仍必须是正方形 PNG。
- 实际 UI 显示尺寸：
  - 顶部状态图标：按可见区域统一视觉尺寸，当前显示盒约 160~250 px。
  - 左下状态/道具图标：22 x 22 px。
  - 右下时间轴/成就图标：24 x 24 px。
- 图标描边沿用原始 PNG；状态填充颜色由 UI 控制，默认 `#D9B96A`，变化时使用绿色/红色。
- 命名示例：`spr_stat_health.png`、`spr_stat_money.png`、`spr_ui_timeline.png`、`spr_ui_achievement.png`。

## 文字规格

- 推荐字体：Microsoft YaHei UI / Microsoft YaHei；缺失时回退 Unity 内置字体。
- 顶部状态名称：22 px，Bold，颜色 `#303230`；显示“身心 / 经济 / 人际 / 学业”，不显示任何具体数值。
- 顶部状态图标：使用统一视觉高度的大图标，深棕描边来自原始 PNG，默认米金填充 `#D9B96A`；通过图标内部自下而上的填充度表达当前数值。
- 拖拽预览圆点：卡片拖入左/右选项范围时，在会受影响的图标上方显示中性深棕圆点；`abs(change) <= 14` 显示小号空心圆，`abs(change) > 14` 显示较大实心圆，不透露正负。
- 提交后数值反馈：玩家完成选择并实际改变数值后，增加项使用绿色 `#6F927E`、减少项使用红色 `#B06A5F`；颜色动画持续 0.75 秒，填充高度用约 1.35 秒平滑过渡。
- 左右选项：34 px，Bold，颜色默认 `#64665F`，拖拽激活时 `#6D8178`。
- 卡片标题：24 px，Bold，居中，颜色 `#303230`。
- 卡片描述：30 px，Regular，居中，颜色 `#303230`，允许自动缩小到 24 px。
- 左下状态/道具文字：20 px，Regular，颜色 `#6E716D`。
- 右下时间轴/成就按钮文字：20 px，Regular，颜色 `#6E716D`；当前版本按钮视觉可见但禁用点击。

## 布局规格

- 顶部状态栏：距顶部约 24 px，左右留白约 190~250 px，四项状态按“身心、经济、人际、学业”排列，每项为“变化圆点 + 填充图标 + 名称”。
- 中央卡片图片：屏幕中心略偏上，显示区域 560 x 420 px。
- 左右选项：位于卡片图片左右两侧，只显示文字，透明点击热区约 430 x 200 px。
- 卡片文字区：位于卡片图片下方，宽约 820 px，高约 190 px。
- 左下状态/道具：距左 88 px，距底 58 px。
- 右下时间轴/成就按钮：距右 88 px，距底 58 px，使用浅米填充和暖灰细描边，当前版本 `interactable = false`。
